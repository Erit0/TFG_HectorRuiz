This repository establishes a workflow from processing the CIC-IDS2017
dataset to generating camouflaged synthetic traffic, designed to evade a Random Forest classifier.

Execution order:
Prueba_GAN+camuflaje.py -> generador_de_trafico.py -> oraculo.py

The remaining files are used for analysis and feature selection.
